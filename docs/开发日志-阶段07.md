## 5. 给后来者的指导

- **启动顺序**：PostgreSQL → Redis → RabbitMQ → Nacos → goods/order/stock → gateway → bi → nginx
- **BI 初始化流程**：
  1. `python bi/data/generate_mock_data.py` — 生成模拟行为数据
  2. `python bi/data/run_offline_pipeline.py` — 训练模型 + 批量预测 + Redis 同步
  3. 访问前端看板"模型训练"标签页可在线重训
- **压测流程**：`python test/jmeter/bi_perf_test.py run`
- **看板集成**：在 App.tsx 中 import BIDashboard 并在导航栏添加"BI看板"标签即可
- **环境变量**：前端需设置 `VITE_BI_BASE_URL=http://localhost:8000`（BI 服务地址）
