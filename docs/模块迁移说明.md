# 模块迁移说明

本次补齐将 `goods/order/stock` 的核心实现收敛到子模块可运行形态，并保留过渡期兼容（`build-helper` 继续引用共享代码）：

- `flashsale-stock`：已从实体占位升级为独立应用，包含 `Controller/Service/Mapper/DTO`。
- `flashsale-order-app`：新增 `StockFeignClient`，下单消费链路优先调用 `stock` 服务扣减库存，失败回退本地 Mapper。
- `flashsale-goods-app`：新增读写路由探针与副本读数据源配置，支持 PostgreSQL 主从演示。

后续如需彻底移除共享源码引用，可按清单逐步下沉 `src/main/java/com/flashsale/flashsale_pro` 到各模块并删除 `build-helper`。
